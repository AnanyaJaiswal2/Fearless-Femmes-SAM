# Fearless Femmes 💜

A smart women safety and menstrual health platform designed to provide predictive insights, emotional tracking, and safety awareness.

---

## 🚀 Features

- 📅 Menstrual cycle tracking & prediction  
- 😊 Daily mood tracking  
- 📊 Emotional pattern insights  
- 🔔 Smart reminders (upcoming cycles, ovulation)  
- 🛡️ Women safety awareness & resources
